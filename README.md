# czimagesort

Lightweight applications to sort images easily.


## Installation

Change into the root directory of this distribution (where the file
`pyproject.toml` is) and run the following command:

```shell
pip install .
```

## Contents

`czbestof`: A tool to sort images based on user preference.

`czcategorise`: A tool to sort images into multiple categories.
